### "למה המוקד" (4 עמודות)

```
1. פיתוח ישראלי
   התכנון והפיתוח נעשו כאן, לא עוד מוצר מיובא עם מדבקה מקומית.

2. הכל תחת קורת גג אחת
   מוצר, התקנה, אחריות ושירות — מגיעים מאותה חברה. יש כתובת אחת לפנות אליה, תמיד.

3. פריסה ארצית וזמינות מהירה
   מוקד המתקינים שלנו פרוס בכל הארץ, עם יעד להתקנה תוך 48–72 שעות מרגע התיאום.

4. שקיפות מחיר מלאה
   המחיר מוצג מראש, כולל התקנה — בלי "השאירו פרטים ונחזור עם הצעה", בלי אותיות קטנות.
```

## עמוד מוצר — מנעול חכם ספציפי

*(שם דגם לדוגמה: "המוקד Home" — placeholder, להחלפה כשייקבע שם דגם סופי)*

### כותרת (H1)

```
המוקד Home — מנעול חכם לדלת הכניסה, עם גיבוי מלא לכל תרחיש
```

### תיאור קצר (פסקת פתיחה)

```
מנעול חכם שפותח בישראל, בנוי לשגרת חיים אמיתית: פתיחה בקוד, באפליקציה או בטביעת אצבע —
וגם כשאין סוללה וגם כשאין אינטרנט, אתם לא נשארים בחוץ. מגיע עם התקנה מלאה על ידי טכנאי מטעם
המוקד ואחריות מורחבת, כך שיש לכם כתובת אחת לכל שאלה גם אחרי הרכישה.
```

### יתרונות מרכזיים (bullets)

```
- גיבוי כפול: פתיחה מכנית וסוללה חיצונית זמינה — גם בהיעדר חשמל או אינטרנט לעולם לא נשארים ננעלים בחוץ.
- ניהול הרשאות פשוט: קוד אישי לכל בן משפחה או עובד, שמתעדכן או מתבטל תוך שניות מהאפליקציה.
- התקנה על ידי טכנאי מטעם המוקד בלבד — לא קבלן חיצוני, אותה חברה שאחראית גם על האחריות והשירות.
- אחריות מורחבת ותמיכה ישירה — כשיש שאלה או תקלה, פונים למוקד אחד, לא ליבואן ואז למתקין ואז לחנות.
```

### הסבר תהליך התקנה

```
אחרי שההזמנה מאושרת, ניצור איתכם קשר לתיאום מועד באותו יום העסקים — היעד שלנו הוא הגעת טכנאי תוך 48–72 שעות
בכל אזור בארץ. הטכנאי שמגיע הוא איש מקצוע מטעם המוקד עצמו, שמתקין את
המנעול על הדלת הקיימת שלכם, בודק שהכל עובד כמו שצריך, ומסביר לכם ולבני המשפחה בדיוק איך
להשתמש בו — כולל מה לעשות במקרה של הפסקת חשמל או תקלה. אתם לא נשארים לבד עם קופסה
וחוברת הוראות.
```

### כותרות למפרט הטכני (ללא מספרים מדויקים — אלה יוזנו בנפרד)

```

- עמידות בתנאי מזג אוויר
- התאמה לסוגי דלתות
- שיטות פתיחה (קוד / אפליקציה / טביעת אצבע)
- עמידות ואבטחה פיזית

## דף נחיתה קמפיין — ממוקד מיכל

### כותרת הירו (ממוקדת-כאב)

```
שמעתם על עוד פריצה בשכונה? שקט נפשי לבית מתחיל במנעול שבודקים אליו כל פרט
```

### תת-כותרת (מפזרת חשש סוללה/אינטרנט)

```
גם בלי אינטרנט וגם בלי סוללה — לעולם לא נשארים ננעלים בחוץ.
המנעול שפיתחנו בישראל, עם התקנה וליווי אישי מהמוקד עצמו.
```

### 3 יתרונות ממוקדים לחששות של מיכל

```
1. אף פעם לא ננעלים בחוץ
   גיבוי מכני וסוללה חיצונית זמינה בכל רגע — גם אם נגמר החשמל או האינטרנט, נכנסים הביתה כרגיל.

2. מי שמתקין הוא מי שאחראי
   הטכנאי שמגיע הביתה הוא איש צוות של המוקד עצמו, לא קבלן זר — אותה חברה שנותנת את האחריות והשירות.

3. כתובת אחת לכל שאלה, גם אחרי הרכישה
   אחריות מורחבת ומוקד שירות זמין — לא צריך "לחפש מישהו" כשיש שאלה או תקלה.
```

### טופס ליד — טקסט מלווה

```
כותרת טופס: קבלו הצעת מחיר מותאמת לבית שלכם
תת-טקסט: השאירו כמה פרטים ונחזור אליכם עם ההמלצה המתאימה — בלי התחייבות.
שדות: שם מלא | טלפון | אזור מגורים | מה הכי חשוב לכם? (למשל: גם לילדים, גיבוי בלי אינטרנט, מחיר)
